This package provides dummy data for first name, last name, full name, gender, birthdate generation based on given age range or random, email and password generation based on given information or random, mobile number generation based on given series number or random, full addresses (including country, state, city, street, address line 1), different cities, states and countries according to chosen country or state, price selection from given range or from given product name and with specific currency, ID generator based specific pre-fix value or next value based on given value or with random values, image selector based on chosen purpose either for profile picture or wallpaper, random hobbies selector, random int or float number generation based on specific range or any value. Random bank and card details selection. OTP generation with different types: Only numbers, numbers and characters, lowercase and uppercase. Feedback text selection based on specific ratings, mood or random data.
